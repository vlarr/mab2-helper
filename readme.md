# Описание

Утилита для упрощения торговли в игре "Mount & Blade II: Bannerlord".

Приложение позволяет сохранять информацию о посещённых городах: список производимых товаров в деревнях принадлежащих
городу, список мастерских в городе, список товаров являющихся избытком или недостатком.

Есть возможность выбрать город как текущий что изменит сортировку по-умолчанию и пометит товары требующие внимания
цветом:

* в начало списка переместится выбранный город,
* далее пойдут города где будет недостаток тех товаров которые в избытке в текущем городе,
* далее идут города в которых избыток тех товаров которые являются недостатком в текущем городе,
* далее идут все прочие города.

## Требования

Необходимы: java 17, maven (apache-maven-3.9.3)

Опционально: docker

Приложением используется порт `8080` по-умолчанию.

## Запуск приложения:

### Как java приложение:

```shell
mvn clean package
cd target
java -jar mab2-helper-{version}.jar
```

### Как docker-приложение 
#### 1) с локальным хранилищем (папка `.\local\db\`) для базы данных:

```shell
mvn clean package
cd compose
docker compose -f .\docker-compose.local-db.yml up
```

#### 2) с подготовленным хранилищем `mab2h-db`:

```shell
mvn clean package
cd compose
docker compose -f .\docker-compose.yml up
```

## Подгрузка данных из примера:

1. Заходим на http://localhost:8080/
2. Выбираем пункт "2: upload city info from json file".
3. Выбираем файл `./examples/city-info-list.json`

4. Нажимаем `Upload`
